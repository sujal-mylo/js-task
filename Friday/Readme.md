# Notes App

Quick notes app I built to organize ideas with tags.

## What it does

- Create notes with a title, body, and tags
- Search through your notes
- Filter by specific tags
- Delete notes you don't need anymore

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## Running the Demo

1. Open `index.html` in browser
2. Show creating a note with tags
3. Show searching functionality
4. Show tag filtering
5. Show deleting a note
6. Open `src/app.js` and explain state management approach
7. Talk about what makes the architecture clean and scalable
